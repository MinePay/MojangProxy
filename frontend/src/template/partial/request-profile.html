GET /v1/profile/{identifier}

Performs a lookup for the profile of a specific player based on their globally unique profile identifier or display name at the current point in time.

Request

Try it yourself:

GET /mojang/v1/profile/{{identifier}} HTTP/1.1
Host: api.minepay.net
Accept: application/json
Response 
HTTP/1.1 200 Ok
Content-Type: application/json;Charset=UTF-8
