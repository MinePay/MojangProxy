GET /v1/profile/{identifier}/history

Performs a lookup for the profile history of a specific player based on their globally unique profile identifier or display name at the current point in time.

Request

Try it yourself:

GET /mojang/v1/profile/{{identifier}}/history HTTP/1.1
Host: api.minepay.net
Accept: application/json
Response 
HTTP/1.1 200 Ok
Content-Type: application/json;Charset=UTF-8

[
    {
        "name":         "LordLorkin",
        "changedToAt":  null
    },
    {
        "name":         "HighLordAkkarin",
        "changedToAt":  1423053791000
    }
]
