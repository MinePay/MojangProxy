GET /

Returns information about the current application version, codename and vendor.

Request 
GET /mojang HTTP/1.1
Host: api.minepay.net
Accept: application/json
Response 
HTTP/1.1 200 Ok
Content-Type: application/json;Charset=UTF-8

{
    "name":     "MCAPI",
    "version":  "0.0.0-SNAPSHOT",
    "vendor":   "Minepay"
}

GET /v1/profile/{identifier}/history

Performs a lookup for the profile history of a specific player based on their globally unique profile identifier or display name at the current point in time.

Request

Try it yourself:

GET /mojang/v1/profile/HighLordAkkarin/history HTTP/1.1
    Host: api.minepay.net
    Accept: application/json
Response 
HTTP/1.1 200 Ok
    Content-Type: application/json;Charset=UTF-8

    [
        {
            "name":         "LordLorkin",
            "changedToAt":  null
        },
        {
            "name":         "HighLordAkkarin",
            "changedToAt":  1423053791000
        }
    ]

GET /v1/name/{name}

Performs a lookup for the profile identifier of a specific player based on their display name at the current point or a specific point in time.

Request

Try it yourself:

GET /mojang/v1/name/HighLordAkkarin HTTP/1.1
Host: api.minepay.net
Accept: application/json
Response 
HTTP/1.1 200 Ok
Content-Type: application/json;Charset=UTF-8

{
    "id":   "d71a5dac4e71443b81584389c269e44d",
    "name": "HighLordAkkarin"
}

POST /v1/name

Performs a lookup for the profile identifiers of a list of player based on their display names at the current point in time.

Request 
POST /mojang/v1/name HTTP/1.1
    Host: api.minepay.net
    Accept: application/json
    Content-Type: application/json

    [
        "HighLordAkkarin",
        "Notch",
        "HopefullyUnregisteredUsername"
    ]
Response 
HTTP/1.1 200 Ok
    Content-Type: application/json;Charset=UTF-8

    [
        {
            "id":   "d71a5dac4e71443b81584389c269e44d",
            "name": "HighLordAkkarin"
        },
        {
            "id":   "069a79f444e94726a5befca90e38aaf5",
            "name": "Notch"
        }
    ]
