GET /

Returns information about the current application version, codename and vendor.

Request 
GET /mojang HTTP/1.1
Host: api.minepay.net
Accept: application/json
Response 
HTTP/1.1 200 Ok
Content-Type: application/json;Charset=UTF-8

{
    "name":     "MCAPI",
    "version":  "0.0.0-SNAPSHOT",
    "vendor":   "Minepay"
}

POST /v1/name

Performs a lookup for the profile identifiers of a list of player based on their display names at the current point in time.

Request 
POST /mojang/v1/name HTTP/1.1
Host: api.minepay.net
Accept: application/json
Content-Type: application/json

[
    "HighLordAkkarin",
    "Notch",
    "HopefullyUnregisteredUsername"
]
Response 
HTTP/1.1 200 Ok
Content-Type: application/json;Charset=UTF-8

[
    {
        "id":   "d71a5dac4e71443b81584389c269e44d",
        "name": "HighLordAkkarin"
    },
    {
        "id":   "069a79f444e94726a5befca90e38aaf5",
        "name": "Notch"
    }
]
